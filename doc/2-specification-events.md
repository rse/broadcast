
Specification
=============

Event Types
-----------

- Townhall (e.g. Digital Townhall)
    - 2500-10000 attendees
    - msg Email addresses
    - Email-Validation
    - Chat/Questions
    - closed group access
    - moderated

- Conference (e.g. Inscom)
    - 200 attendees
    - arbitrary Email addresses
    - Email-Validation
    - Chat/Questions
    - closed group access
    - moderated

- Event (e.g. Open Source Day)
    - 10000 attendees
    - no Email validation
    - public access
    - Chat
    - moderated

- Meeting (e.g. SocietyEXCHANGE)
    - 150 attendees
    - msg Email validation via RegExp
    - Email-Validation
    - Chat/Questions
    - closed group access
    - moderated/unmoderated

- Lecture (e.g. SEIP)
    - 75 attendees
    - TUM Email adresses
    - Email-Validation
    - closed group access
    - Chat/Questions/Emotions
    - unmoderated

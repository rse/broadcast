
#   SPECIFICATION: SOLUTION VISION (SPEC-SV)

✳   Created:  **2026-06-18 10:18**
✎   Modified: **2026-06-18 10:18**

##  ASPECT: Purpose <a id="SPEC-SV-purpose"></a>

msg.Broadcast exists to let organizations stream live video events to thousands of authenticated attendees while keeping
the audience engaged through moderated chat, questions, and likes, all under full control of who may watch and interact.

##  ASPECT: Target Audience <a id="SPEC-SV-audience"></a>

The solution serves event organizers (Managers), live-event moderators and presenters, and large invited audiences of
attendees, ranging from small internal lectures of 75 people up to digital townhalls of 10000 simultaneous participants.

##  ASPECT: Value Proposition <a id="SPEC-SV-value"></a>

msg.Broadcast delivers a privacy-by-design, GDPR-compliant, self-operated streaming platform that combines email-verified
access control, streaming-provider independence, and real-time moderated interaction in a single browser-based product.

##  ASPECT: Differentiators <a id="SPEC-SV-differentiators"></a>

Unlike generic streaming platforms, msg.Broadcast enforces per-attendee access limits, switches streaming providers live
during an event without user interaction, anonymizes all personal data on event finish, and runs cost-efficiently on
self-hosted European infrastructure.

##  ASPECT: Future State <a id="SPEC-SV-future"></a>

Once adopted, msg.Broadcast replaces third-party event platforms as the organization's primary channel for large-scale
live events, providing predictable per-event cost, full data sovereignty, and a consistent, engaging attendee experience.
